# Norden 901 Expedition 2026

A mobile-first, single-page website showcasing the Husqvarna Norden 901 Expedition 2026 motorcycle.

## Design Approach

**Aesthetic Direction:** Industrial-Expedition Brutalism meets Nordic minimalism

- **Typography:** Bebas Neue (display) + Archivo Black (headlines) + Space Mono (body/technical)
- **Color Palette:** Monochromatic with burnt orange accents
- **Key Features:**
  - Dramatic hero section with animated stats
  - Parallax scrolling effects
  - Technical spec grid with hover animations
  - Image galleries with smooth transitions
  - Interactive CTA with mouse tracking
  - Subtle noise texture overlay
  - Mobile-first responsive design

## Technologies

- Pure HTML5, CSS3, and JavaScript (no frameworks)
- Google Fonts (Bebas Neue, Archivo Black, Space Mono)
- CSS Grid and Flexbox for layouts
- Intersection Observer API for scroll animations
- CSS custom properties for theming
